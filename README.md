# SPSI
A Squid3 Proxy Server Installer

---

### Features
* Username & password authentication.
* Black-list any webiste.
* No logs

---

### Usage
   wget -O SPSI "" && bash SPSI [port] [username] [password] && rm SPSI
For example:
   wget -O SPSI "" && bash SPSI 3128 foo bar && rm SPSI
